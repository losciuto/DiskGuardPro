# 🛡️ Disk Guard Pro v1.5.0 - Forensic Platinum
Suite definitiva per l'acquisizione forense e la protezione dati a norma GDPR.

### Caratteristiche Principali
- **Integrità SHA-256**: Calcolo real-time dell'impronta digitale per validità legale.
- **Sicurezza GDPR**: Derivazione chiave PBKDF2 (100k rounds) con Salt casuale.
- **Imaging Forense**: Copia bit-a-bit conforme (inclusi file cancellati e slack space).
- **Software Write Block**: Protezione attiva contro la scrittura sulla sorgente.
- **UX Avanzata**: Velocità di copia in tempo reale ed ETA dinamico.

### Installazione
`pip install -r requirements.txt`
Eseguire con privilegi di Amministratore/Root.
