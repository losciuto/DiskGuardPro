# 🛡️ Disk Guard Pro v1.5.0 - Forensic Platinum
Die ultimative Suite für forensische Datenerfassung und DSGVO-konformen Datenschutz.

### Hauptmerkmale
- **SHA-256 Integrität**: Echtzeit-Fingerprinting für rechtliche Gültigkeit.
- **DSGVO Sicherheit**: PBKDF2-Schlüsselableitung (100k Runden) mit zufälligem Salt.
- **Forensisches Imaging**: Bitgenaue Kopie (einschließlich gelöschter Dateien und Slack Space).
- **Software Write Block**: Aktiver Schreibschutz für das Quelllaufwerk.
- **Erweiterte UX**: Echtzeit-Kopiergeschwindigkeit und dynamische ETA.

### Installation
`pip install -r requirements.txt`
Mit Administrator-/Root-Rechten ausführen.
