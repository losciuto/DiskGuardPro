# 🛡️ Disk Guard Pro v1.5.0 - Forensic Platinum
Suite definitiva para adquisición forense y protección de datos conforme al RGPD.

### Características Principales
- **Integridad SHA-256**: Huella digital en tiempo real para validez legal.
- **Seguridad RGPD**: Derivación de clave PBKDF2 (100k rondas) con Salt aleatorio.
- **Imagen Forense**: Copia bit a bit, incluyendo archivos borrados e "slack space".
- **Software Write Block**: Protección activa contra la escritura en la unidad de origen.
- **UX Avanzada**: Velocidad de copia en tiempo real y ETA dinámico.

### Instalación
`pip install -r requirements.txt`
Ejecutar con privilegios de Administrador/Root.
