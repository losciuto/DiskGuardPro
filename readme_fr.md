# 🛡️ Disk Guard Pro v1.5.0 - Forensic Platinum
Suite ultime pour l'acquisition médico-légale et la protection des données conforme au RGPD.

### Caractéristiques Principales
- **Intégrité SHA-256**: Empreinte numérique en temps réel pour la validité légale.
- **Sécurité RGPD**: Dérivation de clé PBKDF2 (100k rounds) avec Salt aléatoire.
- **Imagerie Forensique**: Copie bit à bit conforme (incluant fichiers supprimés et slack space).
- **Software Write Block**: Protection active contre l'écriture sur le lecteur source.
- **UX Avancée**: Vitesse de copie en temps réel et ETA dynamique.

### Installation
`pip install -r requirements.txt`
Exécuter avec privilèges Administrateur/Root.
