# 🛡️ Disk Guard Pro v1.5.0 - Forensic Platinum
Ultimate suite for forensic acquisition and GDPR-compliant data protection.

### Key Features
- **SHA-256 Integrity**: Real-time fingerprinting for legal validity.
- **GDPR Security**: PBKDF2 key derivation (100k rounds) with random Salt.
- **Forensic Imaging**: Bit-for-bit compliant copy (including deleted files and slack space).
- **Software Write Block**: Active protection against writing to the source drive.
- **Advanced UX**: Real-time copy speed and dynamic ETA.

### Installation
`pip install -r requirements.txt`
Run with Administrator/Root privileges.
