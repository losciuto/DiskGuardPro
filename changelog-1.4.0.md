# 📜 Changelog - Disk Guard Pro

## - Forensic Gold Edition - 2024-05-24
### Aggiunto
- **Software Write Block (SWB)**: Nuova opzione nel tab "Forense" per forzare l'apertura della sorgente in sola lettura (`rb`), impedendo la modifica dei metadati originali.
- **Tab Forense**: Interfaccia dedicata alla gestione dell'integrità delle evidenze.
- **Audit Log Avanzato**: Generazione del file `forensic_audit.csv` per la tracciabilità della catena di custodia.
- **Certificazione Software**: Inserimento dei crediti di conformità forense nel popup Info.

### Migliorato
- **Gestione Errori I/O**: Ottimizzazione della stabilità durante la lettura di dischi con settori danneggiati.
- **UI Scaling**: Migliore adattamento dei widget per schermi ad alta risoluzione.

## - Global Edition
- Supporto multilingua (IT, EN, ES, FR, DE).
- Cambio lingua dinamico e popup info localizzato.

## - GDPR Edition
- Retention Timer e log di conformità CSV.

## - Ultimate Edition
- Mirroring Cloud SFTP e Secure Wipe DoD.

## - Initial Release
- Clonazione byte-to-byte, AES-256 e compressione.
