# 🛡️ Disk Guard Pro v1.4.0 - Forensic Gold

**Disk Guard Pro** è una suite avanzata per l'acquisizione, la protezione e la governance dei dati. Progettata per soddisfare i requisiti della **Digital Forensics** e della conformità **GDPR**, rappresenta lo stato dell'arte per il backup e l'imaging di supporti fisici.

## ⚖️ Caratteristiche Forensi
- **Physical Imaging**: Copia conforme bit-a-bit (include file cancellati e slack space).
- **Software Write Block**: Protezione attiva contro la scrittura accidentale sulla sorgente.
- **Catena di Custodia**: Registrazione di ogni operazione nel file `forensic_audit.csv`.
- **Integrità Certificata**: Hashing SHA-256 e crittografia AES-256 (CTR).

## 🌍 Supporto Internazionale
- **Lingue**: Italiano, English, Español, Français, Deutsch.
- **Interfaccia**: Dinamica e moderna basata su `CustomTkinter`.
